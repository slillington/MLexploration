# MLexploration

Jupyter notebooks and code for exploring ML applications to drug discovery, protein engineering, and AI agents.

## Contents

### `sequence_representations_property_prediction/`
Protein property prediction comparing different sequence representations (one-hot encoding, physicochemical features, protein language model embeddings via ESM2) against multiple model architectures (linear regression, random forests, neural networks) for predicting peptide properties from short amino acid sequences.

### `agent_playground/`
A collection of AI agent projects demonstrating agentic design patterns. Includes a Coursera project implementing the GAME loop (Goals, Actions, Memory, Environment) for autonomous agents, a disease-target intelligence search system, and simple agent orchestration demo.
